\subsubsection*{template$<$class T$>$class S\-Dict$<$ T $>$}

Ordered dictionary of elements of type T. Internally uses a \hyperlink{class_q_list}{Q\-List$<$\-T$>$} and a Q\-Dict$<$\-T$>$.
