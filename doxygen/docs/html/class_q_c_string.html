Descrição detalhada

This is an alternative implementation of QCString. It provides basically the same functions but uses less memory for administration. This class is just a wrapper around a plain C string requiring only 4 bytes "overhead". QCString features sharing of data and stores the string length, but requires 4 + 12 bytes for this (even for the empty string). As doxygen uses a LOT of string during a run it saves a lot of memory to use a more memory efficient implementation at the cost of relatively low runtime overhead.
